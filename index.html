Oppgaver Megatron

Lag en JavaScript-kalkulator for å beregne snitt karakteren din. Multipliser karakterene dine, del på antall karakterer, og få gjennomsnittet.

Bruk en array for å lagre karakterene og en løkke for å beregne summen.

Lag et Node.js RESTful API som tar imot en liste med karakterer og returnerer gjennomsnittet. Gjerne bruk deler av oppgave 1 for oppgaven.

Bruk Express.js for å sette opp API-et og test det med Postman.

Sett opp et prosjekt som bruker GSAP (GreenSock Animation Platform) for å lage scroll-animasjoner. Inkluder instruksjoner for å installere GSAP og implementere en enkel scroll-animasjon.

Start med å installere GSAP via npm og bruk ScrollTrigger-pluginen for å lage animasjoner.
